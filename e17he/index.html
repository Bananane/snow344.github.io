Ever17 Himmel Edition Patch v1.3

Installation

Simply extract the zip file into the game directory.

Font fix

If the font does not look right, install the font located at "_font/lucon.ttf". This will install the Consolas font on your system.

If you run the unpatched version of the game, the font may revert. Overwrite "ever17PC_us.ini" with the patched version if that happens.

Known Issues

  • Entries in the debug menu (eg. character status/flags) might not update immediately. To fix this, enable the last debug entry (System status display).
  • Scene transitions, animations and text will appear faster than the unpatched version. This was done intentionally by raising the FPS cap from 30 to 45 to create a smoother experience. This means that text will sometimes finish before the voiced line, even if voice synchronization is on.
  • Disabling skip mode sometimes results in characters coming out extremely slow (1 character per 3 seconds). This can be fixed by disabling voice synchronization.
  • Saving at a choice then re-loading and looking at the backlog sometimes causes the last line in the backlog to be glitched. Does not affect gameplay.
  • Some segments with Coco/Sara singing lag up the game. This exists in the original and doesn't usually cause a crash.
  • Return to title occasionally crashes. Data will still be saved, but you'll have to start the game again.
  • When returning to title/quitting, the following message appears momentarily: "save system memory". This is purely aesthetic.
  • Alt-tabbing out when the credits are playing and then bringing the game up again sometimes skips the credits.
