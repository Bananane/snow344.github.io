■Problem: Movies and sounds get interrupted unexpectedly.

Please check that you have the latest drivers for your video card
installed in your system.

○Problem: The game is slow or jerky.

Please review the following points:

　・Does your system meet the game requirements (listed in the beginning of this document)?

→Update the video card driver.

　・Do you have an antivirus installed and running?

　　Antiviruses can have a big impact on the game perfomance.
　　It is recommended that you disable them while you play in order to get a better perfomance.

○Problem: Can't start the game at all, or the game closes unexpectedly.

Please review the following points:

　・Does your system meet the game requirements (listed in the beginning of this document)?

　→Update the video card driver.

　・Does your sound card function normally?

　・Are you using one of the supported operating systems?

→This game supports Windows2000/XP/Vista.

　・Do you have any other programs running (including the resident ones)?

→In some cases, programs that run in the background can have a large impact on the game perfomance. It is recommended that you close all other programs including those residing in the Task Tray in order to get a stabler game perfomance.

　・Did you change the install location from its default setting?

→The game might not run correctly if the install path's hierarchy is too complex (too many subdirectories).

　・Are you using any Virtual CD-ROM software?

→This game is not compartible with such software. Please close all Virtual CD-ROM programs before you play. Also, certain CD-R software (like CloneCD) is known to prevent the game from running so it is recommended to uninstall it in order to play the game.

　・Did the game install properly?

→Please reinstall the game. If there any files or folders left from the game after the uninstaller has finished, make sure to remove them manually. Try installing the game into the default install location.

　・Is there any free disk space for the swap file on the drive where your Windows is installed?

→Please make sure there is 1GB or more free space for the swap file.



User Support Center

Phone: 052-779-6549

At the User Support Center, we only provide technical support.
Please note that we do not answer enquiries on game solutions / content and the like.

Before contacting the technical support, it is strongly desired that you could summarize your system environment in the following manner so as to ensure more efficient communication with the staff.

  • Computer maker and model
  • CPU type and speed
  • OS name and (if possible) version number
    (e.g.: WindowsXP SP2)
  • Available memory
  • Video/sound card name and (if possible) driver version number
  • The contents and circumstances of the error message

In order to check whether your system meets the requirements, please use the following method.

    1. Open the Start Menu and click "Run..."
    2. Enter "dxdiag" and click the "OK" button; the DirectX Diagnostics Tool should appear.
    3. Check the "Operating system," "Processor," "Memory," "DirectX version" fields under the "System" tab.
    4. Check the device "Name," "Chip type" and "Available memory" under the "Display" tab.
